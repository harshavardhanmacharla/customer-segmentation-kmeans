# Customer Segmentation using K-Means Clustering

Financial customer behavior analysis using RFM features and K-Means 
clustering on 490K+ transactions — applicable to banking customer 
intelligence and life event detection systems.

## Problem Statement
Identifying high-value customer segments from large-scale transaction 
data to enable targeted business decisions and personalized financial 
services.

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (K-Means Clustering)
- Matplotlib, Seaborn
- Google Colab

## Key Features
- Analyzed 490K+ transaction records
- Engineered RFM (Recency, Frequency, Monetary) features
- Applied Elbow Method to determine optimal number of clusters
- Identified high-value customer segments for business insights

## Real-World Relevance
This project mirrors real banking AI use cases including:
- Customer life event detection from transactions
- Personalized financial product recommendation
- Fraud pattern detection through behavioral clustering

## Results
Successfully segmented customers into distinct behavioral groups,
enabling data-driven targeting strategies.
